# ui-router-scroll
Angular ui-router and ui-scroll

Here we have [ui-scroll](https://github.com/angular-ui/ui-scroll) and [ui-router](https://github.com/angular-ui/ui-router) demo.

==============

Prerequisites:
* install [Git](http://git-scm.com/)
* install [node.js](http://nodejs.org/) with npm (Node Package Manager)

To run:
* clone the repository `git clone https://github.com/dhilt/ui-router-scroll.git`
* `cd ui-router-scroll`
* install nodejs dependencies `npm install`
* run the app server `npm start`
* the app is being opend as `http://localhost:3000/`
